# Spam Email Detection using Machine Learning

## 📌 Overview
This project is a Machine Learning application that classifies emails as **Spam** or **Ham (Not Spam)**. It uses Natural Language Processing (NLP) techniques to process email text and predict whether an email is spam or not.

---

## 🎯 Objectives
- Detect spam emails automatically.
- Improve email security.
- Classify emails with high accuracy.
- Learn the basics of Machine Learning and NLP.

---

## ✨ Features
- Data Cleaning
- Text Preprocessing
- CountVectorizer
- Multinomial Naive Bayes Algorithm
- Model Training and Testing
- Spam Prediction
- User-Friendly Interface

---

## 🛠 Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- NLTK
- Streamlit (Optional)
- Jupyter Notebook

---

## 📂 Dataset
Dataset Name: **mail_data.csv**

Columns:
- Category (Spam/Ham)
- Message

---

## ⚙️ Machine Learning Algorithm
- CountVectorizer
- Multinomial Naive Bayes

## ▶️ How to Run

1. Install Python.
2. Install required libraries.

```bash
pip install pandas numpy scikit-learn nltk streamlit
```

3. Open the project.
4. Run the notebook or app.
5. Enter an email message.
6. Click **Predict** to check whether the email is Spam or Ham.
